Window - AR Game

Description:

For this project, I created an augmented reality game for the quest 3. The timeline for this project was over two semester of my junior year in the GIMM program. Everything was created by me with the assets being modeling in Blender and Maya and textured in Substance Painter. I decided to use the Unity Universal Render Pipeline and also the Meta All in One SDK for development on this project. My game features both hand and controller tracking, haptic feedback, custom music, sound effects, and character animations. The goal for this game was to design and create every single aspect of the game.
